# Digilent Pmod ESP32 SPI communication module for FPGA

Minimal Pmod ESP32 driver to enable low-throughtput wireless communication on FPGA boards.

## Resource Utilization on CMOD-A735T

### Pmod ESP32 Receiver:

| Resource | Utilization |
|----------|-------------|
| LUTs     | 30          |
| FFs      | 41          |
